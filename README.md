# raspi-sysops

A hands-on homelab running Almalinux on a Raspberry Pi 5,
designed to simulate a real enterprise Red Hat Linux environment.

## Goals

- Practice real-world sysadmin and DevOps workflows
- Build a secured, automated, and highly available Linux server.
- Document everything clearly for employers and fellow learners

## Who is this for

Anyone who wants to follow along and build a similar homelab.
ALL scripts and configurations are documented with explanations.

## Project Areas

- System auditing and monitoring
- User and permission management
- Service automation and scheduling
- Security hardening
- Container deployment with Podman
